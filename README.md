# Figura-Cobblemon-GUI
Based on the base script of kcin2001, with new fixes and a handy GUI!

# New features:
• Basic GUI, no need to face the spooky command prompt!

• (Windows) Bundled the script into an .exe, so you don't need to install Python anymore!

• Linux/MacOS support

• Poser checker: checks for the best suited poser for your animation set

• Head path checker: checks for the right path to your "head" group or whatever group you'd prefer

# Bug fixes:
• Fixes HUNDREDS of broken math operations, which would cause syntax errors in some models (shoutouts to Archeops, which had 1404 broken math operations!)

• Fixes broken paths, for models with a cube and group of the same name (such as Mienshao and Charizard)

• Fixes issue where groups with certain names (Head, Body...) would not display in-game (such as Gallade's head)

• Fixes issue where textures/UVs would change resolution during conversion (such as Audino and Bisharp)

• Fixes issue where certain sound effects instructions would crash the avatar (such as wing_flap_medium for flying Pokémon)

# What the script DOESN'T automatically do:
(if any of this applies to you, let us know at the guide sub-forum so we can work through it together!)

• Create missing animations, so if your .bbmodel doesn't have a certain animation it'll t-pose/slide around when doing that action

• Fix missing textures that are not embedded to the avatar (such as Miltank).

• Add a "cry" instruction to Pokémon who don't already have one set up (your "Custom Cry" won't work if this is the case)

• Add "quirks" to your Pokémon

• Add animated textures
